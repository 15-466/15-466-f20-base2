# Tetris 3D

Author: Jianxiang Li, Fengying Yang

Design: a tetris game in 3D space.

Screen Shot:

![Screen Shot](screenshot.png)

How To Play:

Use WASD keys to move the cubes around, and click mouse keys for rotation.
